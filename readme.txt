MBED apps are on mbed.org and can be accessed using credentials from mbed.txt on Dropbox.

 + 1. PC to MBED:	This is program to operate MBED from PC, so kind of maintenance mode.

 + 2. FPGA to Servo: This code is waiting for signal from MBED and after this signal is received, it is sending proper PWM signal to servo to dispense token(s).
